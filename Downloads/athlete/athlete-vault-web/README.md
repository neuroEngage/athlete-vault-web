# AthleteVault Web Platform

## Overview
AthleteVault is India's premier sports talent discovery platform, connecting athletes with colleges, federations, and corporate sponsors.

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Vanilla CSS Modules + Global CSS Variables
- **Language**: TypeScript

## Project Structure
- `src/app/globals.css`: Core design system (Colors, Typography, Utilities)
- `src/app/page.tsx`: Landing Page
- `src/components/`: Reusable UI components (Navbar, etc.)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Design System
The project uses a premium dark mode aesthetic:
- **Primary**: Neon Lime (`#ccff00`)
- **Secondary**: Electric Blue (`#0066ff`)
- **Background**: Dark (`#050505`)
